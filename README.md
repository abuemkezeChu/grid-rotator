## App

![App View](assets/images/image.png)

## About

This is a simple grid rotator app where the grid is rotated either clockwise or anti clockwise depending on the button clicked.

## Build With

- Javascript
- HTML
- CSS

### Prerequisites

Knowledge about JS:

- Basic data structures
- Arrays
- Functions
- Array methods
